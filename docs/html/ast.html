2. Abstract Syntax Tree

2.1. Top Level Declarations

2.1.1. Class Declaration

A class declaration, with its super class and the interfaces it implements

CLASS Person (Object) []
BEGIN
  VAR age: Int
  VAR salary: Int

  FUNC init(age: Int, salary: Int) -> Void
  BEGIN
      self.age = age
      self.salary = salary
  END

  FUNC getAge() -> Int
  BEGIN
      RETURN self.age
  END

  FUNC getSalary() -> Int
  BEGIN
      RETURN self.age
  END
END

2.1.2. Method Declaration

A method declaration includes its function signature and its body

FUNC getAge() -> Int
BEGIN
  RETURN self.age
END

2.1.3. Static Declaration

Idea here is that we want to seperate the static methods and fields from class ones, just like Scala. We have not implemented static fields yet because of recursive dependencies.

STATIC Person
BEGIN
  FUNC compare(p1: Person, p2: Person) -> Int
  BEGIN
      RETURN p1.getAge() - p2.getAge()
  END
END

2.2.2. If Statement

IF (self.salary > 1000) THEN
    self.salary = 1000
ELSE
    self.salary = 500

ifStatement: IF '(' expr ')' THEN stmt (ELSE stmt)?
;

2.2.3. While Statement

WHILE (self.salary > 0)
    self.salary = self.salary - 1
